
# Calculator Project

This is a simple calculator web application built using **HTML, CSS, and JavaScript**.

## Features

* Perform basic calculations (+, −, ×, ÷)
* Clear screen functionality (C button)
* Interactive button clicks
* Clean and simple user interface

## Technologies Used

* HTML
* CSS
* JavaScript
